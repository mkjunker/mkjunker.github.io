Postscript Files

/The following plotting tools are provided as-is. The author makes no guarantees about their accuracy or usefulness. Use them at your own risk./

Don't have a Postscript printer? Try (Ghostscript and Ghostview (GSView for Windows environments). These tools allow viewing and printing Postscript files.

Some of the Postscript files were developed using MetaPost. A Windows version of MetaPost is included with MikTeX at The MikTeX Project Page. For more on MetaPost, try:

  • TUG's MetaPost page
